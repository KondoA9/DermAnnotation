Beta Version: 2.0.2

Updated 2021/07/16

*macOSの場合、起動時に警告が表示されます。Finderからcontrolキーを押下しながら右クリックし、「開く」を選択することで正常に起動でき、以降はこの操作をしなくても起動可能になります。

新しいバージョンがある場合、インターネットに接続していると更新通知が表示されます。プロキシ自動設定(pac)には対応していません。

ベータバージョンには主に以下の変更点があり、動作が不安定な場合があります。
  • 明るさ変更機能を削除
  • 近似色塗りつぶし機能を削除
  • デフォルトペン設定を削除
  • macOSに対応
  • ライト / ダークモードに対応、任意の解像度に対応、全体的にUIを刷新
  • ペン設定のインポート・エクスポート機能を追加
  • 開いた画像にペン設定が存在しない場合、直前に編集していたペン設定を引き継ぐように
  • アノテーション済み画像の保存先を設定可能に
  • 高解像度画像編集時にトリミングが可能に
  • ペイント時の負荷を大幅に低減、高解像度画像の編集がよりスムーズに
  • 高解像度画像編集時、アプリケーションの動作がより安定に
最新リリースの変更点
  • 開いたフォルダごとに出力先フォルダを設定可能になりました。編集画面下の出力欄で個別の出力フォルダが設定でき、設定から規定の出力フォルダを設定可能です。
  • 以前開いたフォルダを開いた際、最後に編集していた画像を開くようになりました。
  • 画像変更時にペンのサイズがリセットされないようになりました。この機能は設定で切替可能です。
  • 画像編集中、しばらくするとペンで塗る際の負荷が増大する問題を修正。
  • スライダーのハンドルが隠れる場合がある問題を修正。
  • 新しいバージョンに更新した際、リリースノートが自動で開かれないように修正。
  • 画像読込中、メニューバーを操作不可に変更。

  • 特定環境下で一部ユーザーインターフェースがずれる問題を修正
