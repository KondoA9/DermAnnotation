Mojave 10.14+

*macOSの場合、アップデート後の初回起動時に警告が表示されます。Finderからcontrolキーを押下しながら右クリックし、「開く」を選択することで正常に起動できます。

バージョン1.x.xから2.x.xへのバージョンに伴い、主に以下の変更点があります。

1.x.xから移行する際、設定やフォルダ履歴等は全て削除されますが、アノテーション済みのファイルはそのまま開くことが出来ます。
ただし、元ファイル－アノテーション済みファイル　の関連付けが削除されるため、作業中のフォルダの作業を全て終わらせた後に移行することをおすすめします。

  • 明るさ変更機能を削除
  • 近似色塗りつぶし機能を削除
  • デフォルトペン設定を削除
  • macOSに対応
  • ペン設定のインポート・エクスポート機能を追加
  • アノテーション済み画像の保存先を設定可能に
  • 拡大 / 縮小の挙動を改善
  • 開いた画像にペン設定が存在しない場合、直前に編集していたペン設定を引き継ぐように機能改善
  • ライト / ダークモードに対応、任意の解像度に対応、全体的にUIを刷新
  • 高解像度画像編集時にトリミングをする機能を追加
  • ペイント時の負荷を大幅に低減、高解像度画像の編集がよりスムーズになるように機能改善
  • 高解像度画像編集時、アプリケーションの動作がより安定になるように機能改善
  • 自動的にバージョンチェック・ダウンロード・バージョンアップが可能に(Windowsの場合警告が表示されるため、手動で警告の対応をする必要があります。)
  • バージョンチェックの際、プロキシ環境下でも更新が可能に
最新リリースの変更点
  • 画像を回転する機能を追加（回転は編集時のみで、出力される画像は回転されません）
  • 一部のアイコンを変更
  • 画像を編集した状態でのアプリ終了時の挙動を改善
  • 通知UIが途中で途切れる場合がある問題を修正
  • ペンの編集時、アプリケーションが強制終了する場合がある問題を修正
  • 戻る・進むをし、その後ペンで編集するとアプリケーションが強制終了する問題を修正
  • フォルダを開いた際、何も表示されない場合がある問題を修正
既知の不具合
  • 戻る / 進むができなくなる場合がある
  • ペンが複数選択されたように表示されてしまう
  • 保存不可能なフォルダを出力先フォルダとして指定すると、アプリケーションがフリーズする場合がある
