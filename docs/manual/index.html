基本的な使い方

フォルダ選択ダイアログで開きたい画像が入ったフォルダを選択します。
編集画面右側のUIで、そのフォルダ内の画像を選択することが出来ます。
開いているフォルダと同じ階層に、画像が入った別のフォルダある場合、そのフォルダに移動することが出来ます。

使用可能な画像形式

  • jpg, png, bmp, gif 等の一般的な画像
  • ndpi, vms, vmu, svs, scn 形式の病理画像
  • DermAnnotationでアノテーション済みの画像

デフォルトのペンを設定する

アプリケーションフォルダ内のdatas/setting/pen.iniを設定することでデフォルトのペンを設定することができます。
[NAME]セクションnameキーにペンの名前をコンマ「,」区切りで記述します。セクション[COLOR_RGB]に、先程のペン名をキーとして、(R, G, B, 255)形式で色を設定します。R, G, Bは0~255の整数で、それぞれ赤、緑、青の濃さに対応します。
例）
[NAME]
name = Color1,Color2
[COLOR_RGB]
Color1 = (255, 0, 0, 255)
Color2 = (0, 255, 0, 255)

ペンを追加/編集する

ペンを右クリックすると、そのペンの名前や色を変更したり、削除することが出来ます。
ペン欄の一番下の「+」アイコンをクリックすると、新しくペンを追加することが出来ます。

別のソフトウェアでDermAnnotation用TIFF画像を扱う

DermAnnotationではTIFF画像にペン情報（名前、色）が保存されており、画像ごとに異なったペンを使用することが可能です。
この情報はTIFFファイルの各ページの名前として、以下のように保存されています。
1ページ目（オリジナル画像）: 任意の名前
2ページ目以降（レイヤ画像）: ペン名/(R, G, B, 255) (ペン名は任意の文字列、R, G, Bは0~255の整数)
他ソフトウェアでは、ページ名を参照することでペン名、色の取得が、ページ名に書き込むことで新たなDermAnnotation用TIFFファイルを作成することができます。
